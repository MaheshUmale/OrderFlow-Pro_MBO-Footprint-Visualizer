**Institutional Grade Order Flow Analysis Tool**

OrderFlow Pro is an advanced financial market visualization suite designed for professional traders. It features a Market-By-Order (MBO) DOM, high-fidelity Footprint Charts, and Bookmap-style Liquidity Heatmaps.

## üåü Features

- **MBO SuperDOM**: Visualize individual order queue positions, detect Iceberg orders (Native, Aggressive, Trapped), and monitor liquidity changes.
- **Footprint Chart**: Detailed auction analysis with Delta, Imbalance, Volume profiles, and customizable aggregation.
- **Liquidity Heatmap**: Historical depth visualization similar to Bookmap to spot trapped traders and absorption.
- **Upstox Bridge**: A specialized Node.js relay server to connect institutional algorithms to live Upstox V3 Market Data (Protobuf).
- **Trade Logic AI**: Real-time signal generation including Absorption, Momentum, Structure Breaks, and CVD Divergences.

## üõ†Ô∏è Installation & Setup

### 1. Install Dependencies

You need to install dependencies for both the React Frontend and the Node.js Bridge.

```bash
# Install root dependencies
npm install

# Install Bridge dependencies
npm install ws protobufjs upstox-nodejs axios
```

### 2. Configure Upstox Bridge

1.  Download `market_data_feed.proto` from the [Upstox Node.js SDK examples](https://github.com/upstox/upstox-nodejs/blob/master/examples/websocket/market_data/v3/market_data_feed.proto).
2.  Place the file in the **root** directory of this project.

### 3. Run the Application

**Start the Backend Bridge:**
```bash
node server/upstox-bridge.js
```
*Server runs on `ws://localhost:4000`*

**Start the Frontend:**
```bash
npm start
```

## üìà Usage

1.  Open the web application.
2.  Click **Connect Live** in the top control panel.
3.  Enter your **Upstox Access Token**.
4.  The application will begin streaming live tick data, generating Footprint bars and Heatmap layers in real-time.
